# happy_backend

Happy é uma aplicão para ajudar as pessoas a encontrar orfanatos  
proximos em sua cidade e ver informações, detalhes de visitação,  
regras, dias e horário de visitas, contato etc.

## Tecnologias utilizadas

- Nodejs  
- Express  
- Typescript  
- Typeorm  
- PostgressSQL

## Uso
com **npm**  
 `npm install`  
 `npm run dev`  
 
 ou com **yarn**  
 `yarn`  
 `yarn dev`
